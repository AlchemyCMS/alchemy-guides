## Authentication

### alchemy-devise <Badge type="tip" text="official" />

Devise-based user authentication for AlchemyCMS. If you don't bring your own authentication, you can use this gem to authorize users in the Alchemy admin.

<ExtensionLinks gem="alchemy-devise" github="https://github.com/AlchemyCMS/alchemy-devise" />

## E-commerce

### alchemy-solidus <Badge type="tip" text="official" />

Integrates [Solidus](https://solidus.io) e-commerce with AlchemyCMS. Provides shared admin navigation between both backends, ingredients for placing Solidus products, variants, and taxons on Alchemy pages, and shared admin sessions and user abilities.

<ExtensionLinks gem="alchemy-solidus" github="https://github.com/AlchemyCMS/alchemy-solidus" />

## API

### alchemy-json_api <Badge type="tip" text="official" />

A [JSON:API](https://jsonapi.org) compliant API for AlchemyCMS. Use Alchemy as a headless CMS and consume content from any frontend framework.

<ExtensionLinks gem="alchemy-json_api" github="https://github.com/AlchemyCMS/alchemy-json_api" />

## Search

### alchemy-pg_search <Badge type="tip" text="official" />

PostgreSQL full text search for AlchemyCMS. Adds search functionality using PostgreSQL's built-in full text search capabilities.

<ExtensionLinks gem="alchemy-pg_search" github="https://github.com/AlchemyCMS/alchemy-pg_search" />

## Storage

### alchemy-dragonfly-s3 <Badge type="tip" text="official" />

Store Alchemy pictures and file attachments on Amazon S3 using the [Dragonfly](https://markevans.github.io/dragonfly/) S3 datastore.

<ExtensionLinks gem="alchemy-dragonfly-s3" github="https://github.com/AlchemyCMS/alchemy-dragonfly-s3" />

### alchemy_cloudinary <Badge type="tip" text="official" />

Render AlchemyCMS images directly from [Cloudinary](https://cloudinary.com) with smart resizing, cropping, and CDN delivery.

<ExtensionLinks gem="alchemy_cloudinary" github="https://github.com/AlchemyCMS/alchemy_cloudinary" />
