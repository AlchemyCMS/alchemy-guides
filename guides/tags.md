# Tags

Tags help organize pictures and attachments in the [library](library), [pages](pages), and [taggable elements](elements#tagging). Editors can assign tags when uploading or editing media, and use them to filter the library. Alchemy uses [Gutentag](https://github.com/pat/gutentag) as its tagging engine.

## Managing Tags

The Tags module in the admin interface lets you:

- View all tags with their usage count
- Rename tags
- Merge duplicate tags into one
- Delete unused tags

## Merging Tags

When editing a tag, you can merge it into another existing tag. All pictures and attachments tagged with the old tag are re-tagged with the new one, and the old tag is removed. This is useful for cleaning up duplicates or consolidating related tags.
